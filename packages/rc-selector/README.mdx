# @alicloud/console-components-selector

Selector 是 Select 的一个增强组件，具备下拉触底翻页加载数据的能力。可结合 Field 使用。Selector 虽然是 Select 的一个增强，但并未完全覆盖 Select 的使用场景，在 Select 可满足使用的情况下，请优先选择使用 Select。
